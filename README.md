# IBM Cloud AI Internship Project

## AI-Based Cloud Resource Monitoring System

This project demonstrates a simple AI and cloud-based monitoring system developed using Python.

### Features
- CPU usage monitoring
- Memory usage monitoring
- Real-time monitoring
- Beginner-friendly project

### Technologies Used
- Python
- IBM SkillsBuild
- IBM Cloud Concepts
- GitHub

### Run the Project

Install requirements:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```
